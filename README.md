<h1><p style="text-align:center; color:blue;">Do que se trata esse repositório?</p><h1>

Este é um repositório criado para *guardar anotações dos estudos de algumas linguagens de programação*, como C e Python. 
No momento, estamos lançando anotações de estudos da *linguagem C*, com os livros ❝The C Programming Language 2nd Edition❞ e ❝Modern C❞. 
*As anotações têm outras fontes e também são revisadas pelo ChatGPT, sinta-se a vontade para usar essas anotações em seus estudos!*

<h1>Arquivos<h1>

  <h3>C versão C23</h3>
  
  Anotações do livro ["Modern C Third Edition"](C/Modern C):
  Até o momento não iniciamos a leitura.

  <h3>C versão C89/C90</h3>

  Anotações do livro "The C Programming Language"(mesmo que na sintaxe antiga, é um clássico da linguagem):
  
  1.1 Começando
    main() — função principal do programa;
    printf() — imprime texto formatado na tela;
    Estrutura básica de um programa em C;
    Compilação e execução.

  1.2 – Variables and Arithmetic Expressions
    Declaração de variáveis (int, float);
    Expressões aritméticas (+, -, *, /);
    Conversão automática de tipos (inteiros para float).

  1.3 – The for Statement
    Laço for — repetição com contador;
    Incremento e controle de fluxo.
  
  1.4 – Symbolic Constants
    #define — criação de constantes simbólicas;
    Boas práticas para evitar números “mágicos” no código.

  1.5(e subcapítulos) – Character Input and Output
    getchar() — lê um caractere do teclado (ou do input);
    putchar() — escreve um caractere na tela;
    EOF (End Of File) — valor especial que indica o fim da entrada;
    Introduz uso de estados (IN, OUT).

  1.6 – Arrays
    Declaração e uso de arrays (vetores).

  1.7 – Functions
    Definição de funções próprias;
    Passagem de parâmetros;
    Valor de retorno (return).

  1.8 – Arguments – Call by Value
    Argumentos;
    Passagem por valor.

  1.9 – Character Arrays
    Strings em C (arrays de caracteres);
    '\0' (null character);
    Funções auxiliares.

  1.10 – External Variables and Scope
    Variáveis globais (externas);
    Diferença entre escopo local e global;
    Exemplo com funções compartilhando dados globais.

Os outros conteúdos do livro não serão anotados. Após a conclusão faremos anotações do "Modern C Third Edition".
