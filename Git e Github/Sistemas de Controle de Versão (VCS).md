# **Oque São?

Os sistemas de controle de versão controlam as versões de um arquivo ao longo do tempo, gerenciam quais foram as alterações, quem as fez, em que data as fez...Isso deixa o código organizado, controlado e seguro.

# **Tipos

Dentre os VCS, temos os *VCS Centralizado (CVCS)* que possui apenas um servidor central que armazena todos os arquivos das versões e os *VCS Distribuído (DVCS)*, que clona o repositório (servidor central) localmente, o que possibilita um fluxo de trabalho flexível (Por exemplo, mesmo que o servidor central caia, ainda é possível trabalhar no código).
