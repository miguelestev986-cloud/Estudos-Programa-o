# **Importância

A indentação no código o torna mais legível e manutenível. Mas em Python, ela também exerce um segundo papel: Através da indentação o interpretador consegue determinar onde um bloco de comando inicia e onde termina. 

# **Boas Práticas

É uma convenção que a cada novo bloco de código adicionamos 4 espaços. Exemplo:

```python
nome_curso = input('Nome do curso: ')
if nome_curso == 'Python':
    print('Ótima escolha!')
else:
	print('Legalzinho até')
```
